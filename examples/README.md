# Reproduced Examples

The two scripts below reproduce the motivating examples in Section 6:

```console
python code/verify_append_gain_failure.py
python code/verify_diagonal_transfer_failure.py
```

Example 6.1 shows that an append-gain polynomial need not be log-concave.
Example 6.2 shows that the diagonal transfer analogue can fail. These examples
motivate the off-diagonal continuation-transfer state; they are not inputs to
the proofs.
